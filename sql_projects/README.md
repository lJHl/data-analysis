## Настройки

В модуле `settings/db_config.py` добавить настройки:  
`USER` – имя пользователя  
`PWD`&nbsp;&nbsp; – пароль  
`HOST` – хост  
`PORT` – порт подключения  
`DB` &nbsp;&nbsp;&nbsp; – название базы данных
